**语音活动检测（Voice Activity Detection，VAD）** 是语音信号处理中的关键环节，其核心功能是从连续的音频流中准确识别并分割出有效语音段，同时排除静音、噪声等非语音部分。 这一技术直接影响语音识别、语音交互、通信降噪等系统的效率与用户体验。

VAD的核心任务包括语音端点检测，即精确判断语音的起始点和结束点，从而实现对有效语音的截取。 在技术实现上，VAD方法经历了从传统信号处理到深度学习的演进：

- 基于阈值的方法：通过分帧处理（帧长通常为20-30ms，帧移10ms），计算每帧的短时能量（STE）和零穿越率（ZCR），并设定双阈值来区分语音与非语音段。 这种方法简单高效，但对突发噪声敏感，且阈值需手动调整。
- 基于统计模型的方法：如高斯混合模型（GMM），通过建模语音和噪声的MFCC特征分布，计算帧属于语音的后验概率进行判定。 该方法适应非平稳噪声，但依赖大量标注数据。
- 基于深度学习的方法：例如CRNN（卷积神经网络-循环神经网络）模型，输入对数梅尔谱图，利用CNN提取频谱特征，RNN建模时序依赖，输出每帧的语音概率。 深度学习模型具有高精度和强鲁棒性，但需要GPU加速。

在优化策略方面，VAD常结合噪声抑制预处理（如谱减法或深度学习降噪）以提升噪声环境下的性能。 后处理技术如中值滤波可消除短时误判，而挂起与结束检测机制能避免语音尾部过早截断。 工程实践中，需根据场景选择轻量级模型（如MobileNetVAD）以平衡实时性与资源消耗。

VAD的应用场景广泛，典型应用场景如下：

- 实时通信：视频会议静音检测、VoIP通话质量优化
- 语音助手：降低误唤醒率（如唤醒词前的语音过滤）
- 音频预处理：语音识别（ASR）前的噪音过滤
- 边缘计算：嵌入式设备（如智能音箱、车载系统）
- 数据分析：通话录音自动分段、客服语音质检
